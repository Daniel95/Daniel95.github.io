Summary

This is a horror game targeted for the Oculus Rift. The goal of the game is to reach the end of the level without being killed by the monster that patrols the level. You can use mana to either light the path before you, or permantently enlighten candles all around the level.

Game features:

-Big map that you can move through
-Defend resource spots for more money
-Place and upgrade towers 3 different towers
-dynamic wave system that spends points
-3d sound effects
-lightning effects
