Summary

Tower defense game where you have to defend the treehouse by placing towers,
upgrading them with resources, and buy and defend resource spots to earn more resources!

Game features:

-Big map that you can move through
-Defend resource spots for more money
-Place and upgrade towers 3 different towers
-dynamic wave system that spends points
