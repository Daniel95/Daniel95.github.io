Summary

This is going to be an action game for mobile, with possibly some endless runner
elements like a randomized level!

I started this project on 18/11/2016.
Once it is fully fleshed out i will release it for mobile.

The player moves and turns automaticly, but you can manipulate its movement.
The arrows on the player points towards the direction you are going to move next,
if possible.

This is game is in an early phase of development, expect bugs.

Game features:

-Grappling Hook
-Automatic Movement
-Procedurally generated Level (coming soon!)
